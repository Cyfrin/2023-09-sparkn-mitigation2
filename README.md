# How to Start
1. Install dependencies
```bash
$ forge install
```

2. Run tests
```bash
$ make test
```

3. Deploy contracts
```bash
$ forge deploy --network <network>
```

or deploy to local network
```bash
make deploy
```

4. Other things to do

Format the codes
```bash
$ make format
```

Clean up the repository
```bash
$ make clean
```
